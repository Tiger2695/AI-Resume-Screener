# 🤖 AI Resume Screener

Production-ready RAG-based resume screening tool.
**AI-powered resume screening tool for HR teams**

## Features

✅ PDF resume parsing
✅ Gemini embeddings + ChromaDB vector search
✅ AI-powered candidate evaluation
✅ Excel report download

